# alexa-teatimer
*Alexa, start tea timer*


## Lambda functions
- Go to *src/* and use `make lambda`.
- Upload *src/alexa-tea-timer.zip* as lambda function in your AWS console.

## Alexa Skill
Files are stored in the *speechAssets* folder
